What is the document Object model (dom)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a web page as a tree of objects that can be manipulated by programming languages like JavaScript. Each element of the webpage (like headings, paragraphs, buttons, etc.) becomes a node in this tree

What is event delegation in the context of the DOM, and why is it useful?

Event Bubbling: When an event occurs on an element, it first runs any handlers on that element, then "bubbles" up to its parent, and so on up to the root of the DOM. By listening on the parent, you can capture events from any of its child elements. Using target Property: The event object has a target property that tells you which child element triggered the event. This allows you to determine which child element was clicked or interacted with, even though the listener is on the parent.o
